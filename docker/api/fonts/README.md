# Fonts used on Southeast & Midwest Conservation Blueprint Explorer reports

Fonts were downloaded manually from Google Fonts:

- https://fonts.google.com/specimen/Raleway
- https://fonts.google.com/specimen/Source+Sans+3
- https://fonts.google.com/specimen/Montserrat

The subset of font styles used in the report were copied to this folder.

See the `LICENSE-<font>.txt` for each font in this folder for their respective
licenses.
